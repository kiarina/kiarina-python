# Changelog

All notable changes to the kiarina-agi-video package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [2.15.0] - 2026-07-14

### Added

- Add a kiapi video generation provider with the LTX-2 family.

### Changed

- Replace MoviePy file video decoding with direct FFmpeg processing through imageio-ffmpeg.

## [2.10.0] - 2026-07-09

### Added
- Add the `kiarina-agi-video` package.

### Changed
- Remove the OpenAI video generation provider.
